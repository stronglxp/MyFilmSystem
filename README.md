## 1、相关技术

#### 前端技术：BootStrap + Thymeleaf

#### 后端技术：SpringBoot + MyBatis

#### 权限管理：Shiro

#### 图片存储：七牛云

#### 数据库：MySQL

## 2、系统页面

系统分为两种角色：商家和顾客，预留了管理员角色的接口，使用Shiro来管理角色。图片存储在七牛云。
